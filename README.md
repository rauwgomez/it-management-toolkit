# IT Project Manager Toolkit

A practical reference for IT Project Managers and IT Managers. This repo holds templates you can copy into real projects, short notes from free courses, and examples of common deliverables.

---

## Who this is for

- IT Project Managers who want reusable artifacts without hunting through Google Docs
- IT Managers transitioning into formal project management
- Anyone following the [10-week learning roadmap](ROADMAP.md) to build skills across cloud, networking, security, and Agile tooling

---

## What's inside

```
templates/         Ready-to-use documents you fill in for real projects
examples/          Filled-in samples so you can see what "done" looks like
notes/             Short study notes from free courses (IBM, Atlassian, Cisco, Microsoft, Fortinet)
certificates/      Place to store PDFs or screenshots of completed course certificates
ROADMAP.md         10-week self-study plan with free resources and weekly goals
```

### Templates

| File | What it is |
|---|---|
| `templates/project-charter.md` | One-page charter: scope, objectives, stakeholders, constraints |
| `templates/weekly-status-report.md` | Weekly update to share with stakeholders |
| `templates/raid-log.md` | Risks, Assumptions, Issues, Dependencies tracker |
| `templates/incident-postmortem.md` | Post-incident review for IT outages or failures |
| `templates/change-request.md` | Formal change request for controlled environments |
| `templates/sla-sli-slo.md` | One-pager defining service levels for a system or team |
| `templates/stakeholder-communication-plan.md` | Who gets what information, how often, and how |

---

## How to use this

1. **Clone or fork** the repo so you have your own copy.
2. **Pick a template** from `templates/` and copy it into your project folder or a Google Doc.
3. **Fill in the placeholders** — anything in `[square brackets]` is meant to be replaced.
4. **Check the examples/** folder if you're unsure what a completed document looks like.
5. **Follow the roadmap** in `ROADMAP.md` to build knowledge alongside the templates.

## Learning roadmap

See [ROADMAP.md](ROADMAP.md) for a 10-week free study plan covering:

- Agile and project management fundamentals (IBM SkillsBuild)
- Jira and Atlassian tools (Atlassian University)
- Networking and cybersecurity basics (Cisco NetAcad)
- Azure, DevOps, and security fundamentals (Microsoft Learn)
- Network security awareness (Fortinet NSE 1 & 2)

All resources are free. The plan is roughly 5–8 hours per week.
