# Onboarding / Offboarding Checklist (IT)

## Onboarding
### Accounts & access
- [ ] Corporate email created
- [ ] MFA enabled
- [ ] SSO/identity group assigned
- [ ] VPN access (if required)
- [ ] Jira/Confluence access
- [ ] Repo access (GitHub/GitLab)
- [ ] Service desk access (if applicable)

### Device
- [ ] Laptop provisioned
- [ ] Disk encryption enabled
- [ ] Endpoint protection installed
- [ ] Required software installed

### Basics
- [ ] Security awareness briefing
- [ ] Access follows least privilege
- [ ] First-week checklist shared

## Offboarding
### Access removal
- [ ] Disable account / remove SSO access
- [ ] Revoke VPN
- [ ] Remove from groups
- [ ] Rotate shared secrets (if any)
- [ ] Transfer ownership of docs/repos

### Device and data
- [ ] Device returned
- [ ] Data retention requirements checked
- [ ] Mailbox/doc ownership reassigned

### Audit trail
- [ ] Ticket created and closed with approvals logged
