# 10-Week Learning Roadmap

This plan covers the core skills an IT Project Manager or IT Manager needs: project delivery, Agile tooling, networking basics, cloud fundamentals, and security awareness. Every resource listed is free.

**Time commitment:** roughly 5–8 hours per week.

**How to track progress:** copy this file, check off items as you finish them, and drop completed certificates in the `certificates/` folder.

---

## Week 1 — Project Management Foundations

**Platform:** IBM SkillsBuild

| Task | Resource |
|---|---|
| Complete course | [IT Project Management Fundamentals](https://skillsbuild.org/learn) |
| Deliverable | Fill in `templates/project-charter.md` with a real or practice project |

**Goal:** understand project phases, stakeholder roles, and the purpose of a charter.

---

## Week 2 — Agile Fundamentals

**Platform:** IBM SkillsBuild

| Task | Resource |
|---|---|
| Complete course | [Agile Explorer](https://skillsbuild.org/learn) |
| Deliverable | Write a half-page note in `notes/agile-basics.md` summarising Scrum roles and ceremonies |

**Goal:** understand the difference between Waterfall and Agile, and when each fits.

---

## Week 3 — Jira for Project Tracking

**Platform:** Atlassian University

| Task | Resource |
|---|---|
| Complete course | [Jira Fundamentals](https://university.atlassian.com) |
| Deliverable | Set up a free Jira project and create a sample backlog with epics, stories, and tasks |

**Goal:** be able to create and manage a Jira board for a project you run.

---

## Week 4 — Atlassian Cloud and ITSM

**Platform:** Atlassian University

| Task | Resource |
|---|---|
| Complete course | [Atlassian Cloud Foundations](https://university.atlassian.com) |
| Optional | [Jira Service Management Fundamentals](https://university.atlassian.com) |
| Deliverable | Fill in `templates/sla-sli-slo.md` for a hypothetical or real service |

**Goal:** understand how Atlassian Cloud products fit together; get a basic grasp of ITSM concepts (SLAs, request queues, incidents).

---

## Week 5 — Networking Basics

**Platform:** Cisco NetAcad

| Task | Resource |
|---|---|
| Complete course | [Networking Basics](https://www.netacad.com) |
| Deliverable | Write a one-page note in `notes/networking-basics.md` covering IP addressing, DNS, and common ports |

**Goal:** be able to hold a conversation with your network team and understand the impact of network issues on projects.

---

## Week 6 — Cybersecurity Awareness

**Platform:** Cisco NetAcad

| Task | Resource |
|---|---|
| Complete course | [Introduction to Cybersecurity](https://www.netacad.com) |
| Deliverable | Fill in `templates/incident-postmortem.md` using a public breach case study as a practice scenario |

**Goal:** understand common threats (phishing, ransomware, data breaches) and why security belongs in every project plan.

---

## Week 7 — Azure Cloud Fundamentals

**Platform:** Microsoft Learn

| Task | Resource |
|---|---|
| Complete learning path | [Azure Fundamentals (AZ-900 path)](https://learn.microsoft.com/en-us/training/paths/az-900-describe-cloud-concepts/) |
| Deliverable | Add a note in `notes/azure-fundamentals.md` listing the core Azure services relevant to IT project delivery (compute, storage, networking, identity) |

**Goal:** understand cloud concepts well enough to scope cloud-related projects and ask the right questions of your technical team.

---

## Week 8 — DevOps Fundamentals

**Platform:** Microsoft Learn

| Task | Resource |
|---|---|
| Complete learning path | [DevOps Foundations](https://learn.microsoft.com/en-us/training/paths/devops-foundations-core-principles-practices/) |
| Deliverable | Write a short note in `notes/devops-fundamentals.md` explaining CI/CD, version control, and how they affect release planning |

**Goal:** understand how DevOps practices change project delivery timelines and handoffs.

---

## Week 9 — Security Fundamentals

**Platform:** Microsoft Learn + Fortinet Training Institute

| Task | Resource |
|---|---|
| Complete learning path | [Security Fundamentals (SC-900 path)](https://learn.microsoft.com/en-us/training/paths/describe-concepts-of-security-compliance-identity/) |
| Complete course | [Fortinet NSE 1 – Information Security Awareness](https://training.fortinet.com) |
| Optional | [Fortinet NSE 2 – The Evolution of Cybersecurity](https://training.fortinet.com) |
| Deliverable | Fill in `templates/change-request.md` for a hypothetical security patch deployment |

**Goal:** understand identity, compliance, and threat landscape concepts; earn NSE 1 (and optionally NSE 2) badges.

---

## Week 10 — Consolidate and Publish

**No new courses this week.** Use the time to wrap up anything incomplete and polish your portfolio.

| Task | Notes |
|---|---|
| Finish any incomplete courses | Check your progress on each platform |
| Add certificates to `certificates/` | PDFs or screenshots work fine |
| Fill in remaining templates | `templates/weekly-status-report.md`, `templates/raid-log.md`, `templates/stakeholder-communication-plan.md` |
| Add one real or practice example to `examples/` | Pick the template you're most likely to use in work |
| Review your GitHub profile | Make sure the repo description and pinned repos look professional |

**Goal:** have a clean, complete portfolio repo you can point to in job applications or interviews.
